# MSIX 安装指南

如果你下载的是 `coriander_player.msix` 安装包，请按照以下步骤操作：

1. 下载 `coriander_player.msix` 后**右键**并进入**属性**页面
2. 在顶栏选择"**数字签名**"页
3. 双击"签名列表"中的第一项（或单击选中后点**详细信息**按钮）
4. 在弹出的"数字签名详细信息"页点击**查看证书**按钮
5. 在弹出的证书页点击**安装证书**
6. 证书导入向导中：
   - 存储位置：选择"**本地计算机**"，下一步
   - 选择"**将所有的证书都放入下列存储**"，点击右侧的**浏览**按钮，选择"**受信任的根证书颁发机构**"，确定，下一步
7. 双击 `coriander_player.msix` 安装

> **提示**：安装证书只需执行一次，后续更新直接双击 MSIX 即可。
