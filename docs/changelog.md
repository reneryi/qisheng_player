# 更新日志 (Changelog)

本文件记录 **栖声播放器 (Qisheng Player)** 的主要变更与更新历史。格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

---

## [1.0.0] - 2026-04-29 - 栖声品牌化与全面升级

本项目自上游 [coriander_player](https://github.com/Ferry-200/coriander_player) Fork 后，经历了深度的二次开发与优化。`1.0.0` 是栖声品牌化后的第一个完整正式版本，标志着播放器在视觉交互与稳定性上的全新起点。

### 🌟 亮点与新增特性

- **品牌重塑**：将应用正式命名整合为 `qisheng_player` (栖声)，全新设计的圆形应用图标与安装包命名。
- **现代化视觉重构**：全面引入 **玻璃拟态 (Glassmorphism)** 视觉语言，统一主界面、底部播放栏、侧边栏、页面面板的毛玻璃效果。
- **Apple Music 风格动效**：
  - 动态流体背景：背景随当前播放专辑的主色调缓慢弥散流转。
  - Hero 共享元素转场：专辑页到详情页的封面图片实现无缝飞跃动效。
  - shared-axis 页面切换：侧边栏切换时加入平滑的淡入淡出过渡。
- **沉浸式 Now Playing (正在播放) 页面**：
  - 支持沉浸式与专业式两种布局模式，保留精美的竖向/横向歌词。
  - 增强逐字歌词动效：当前行支持字号、字重放大及透明度与发光变化。
  - 右侧歌词预览栏：音乐页新增可悬浮展开的迷你歌词栏，随时查看滚动歌词。
- **增强型曲库与文件夹管理**：
  - 文件夹管理迁移至独立全屏页面，操作更加直观。
  - 完善 A-Z/拼音索引、当前播放快速定位、多维度排序及灵活的视图切换。
- **专业级音频控制**：
  - 接入 BASS 库扩展，暴露 FFT 频谱数据，为未来全局音频可视化打下基础。

### 🛠️ 改进与优化

- **性能优化**：引入 `UiEffectsLevel` 效果档位，默认 `balanced` 模式下智能降低模糊成本与动画帧率，有效消除主界面卡顿。
- **布局微调**：
  - 侧边栏改造为悬浮胶囊形态，选中项增加动态强调色。
  - 优化顶部导航与标题栏：新增独立后退/前进按钮，动作按钮更贴近右侧，视觉层级分明。
  - 底部播放器重排：随机播放、循环模式拆分为独立控制按钮，音量调节升级为胶囊滑杆。
- **智能取色与对比度**：动态专辑取色现仅影响强调色与背景氛围，确保了文本的高对比度和可读性。

### 🐛 问题修复

- **彻底消除中文乱码**：
  - 修复底层 Rust 索引时常见的 UTF-8 / Latin-1 / Windows-1252 解码冲突引发的日韩中文乱码。
  - 补全 Dart 层的显示兜底机制，自动清理残留控制字符和 `锟斤拷`，无名文件统一显示为“未知艺术家/专辑”。
  - 修复应用内遗留的英文/乱码文本（如“文件夀”、“ReplayGain”翻译修正）。
- **稳定性修复**：
  - 修复左侧栏快速切页时因 `AnimatedSwitcher` `GlobalKey` 冲突导致的 UI 卡死。
  - 修复播放器最大化切换时，音量/进度 Slider 宽度坍缩到 0 导致的程序崩溃。
  - 修复同步歌词进度插值错误，确保逐字高亮精准同步。
  - 修复部分旧版本偏好设置引发的异常，增强配置向后兼容性。

## 历史版本说明

`1.0.0` 版本整合了旧的 `1.7.x` (原 coriander_player 迭代记录) 预发布记录。旧版本的核心功能如多文件夹扫库、全局快捷键、CUE 完美分轨支持等均在 `1.0.0` 中得到了保留和深度优化。

后续版本将继续遵循语义化版本号 `1.x.y` 进行稳定迭代。
