# Qisheng Player v1.1.0

发布日期：2026-05-01

本版本聚焦界面一致性、桌面歌词可靠性、启动体验和发布更新链路。升级后，播放器会在后台静默更新曲库索引，并在发现 GitHub 新版本时弹出更新提示。

## 亮点

- 启动时自动检查 GitHub Release；发现新版本时提示用户前往 GitHub 下载。
- 修复桌面歌词未随主程序打包和 Debug/Release 路径不一致导致的“桌面歌词未找到”。
- 启动自动扫库改为后台静默执行，不再显示扫描文件夹界面。
- 页面切换时旧界面增加模糊淡化效果，过渡更自然。
- 顶栏和主界面 UI 进一步收敛，圆角、按钮层级和歌词动画更统一。

## 改进

- 去掉顶栏歌词/搜索区域和窗口控制按钮组外层重复玻璃框。
- 歌曲列表“更多”按钮改为嵌入式显示，减少视觉突兀。
- 修正字母导航栏选中字母偏下的问题。
- 优化顶栏歌词切换动画，减少硬切感。
- Windows 打包脚本会自动构建并校验桌面歌词 bundle。
- CI 主程序产物会包含 `desktop_lyric` 子目录。

## 修复

- 修复设置页“系统与工兀”乱码。
- 修复桌面歌词查找路径只覆盖安装目录、未覆盖 Debug 运行场景的问题。
- 修复启动时文件夹数量变化会主动显示扫描进度界面的问题。

## 发布产物

- `Qisheng-Player-v1.1.0-Windows-x64.zip`
- `Qisheng-Player-v1.1.0-Setup-x64.exe`

## 升级提示

- 如使用便携版，请完整解压 zip，确保 `desktop_lyric/` 子目录与 `qisheng_player.exe` 位于同一目录。
- 播放器暂不支持应用内自动替换安装包；更新提示会打开 GitHub Release 页面下载新版。
