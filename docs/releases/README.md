# 版本发布说明

此目录用于保存与版本发布相关的说明文档和发布 payload。

## 目录约定

- `v<version>.md`
  - 面向人阅读的版本发布说明。
- `v<version>.json`
  - 面向发布工具或自动化流程的 release payload。
- `archive/`
  - 历史遗留说明文件归档。
