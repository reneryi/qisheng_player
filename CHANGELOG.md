# 更新日志

本文件记录项目的所有显著变更。格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.6.2] - 2026-04-18

### 修复

- 修复播放状态偶发不同步问题：持续播放时，播放详情页与迷你播放器不再出现“实际在播但按钮显示播放三角形”的错位
- 修复重复启动会新开进程窗口的问题：从任务栏图标或快捷方式再次启动时，改为唤醒已运行实例（含托盘隐藏状态）

### 改进

- 托盘图标改为启动即常驻显示，不再仅在关闭窗口后出现
- 托盘恢复逻辑优化：恢复主窗口时保留托盘图标，退出应用时再统一移除

## [1.6.1] - 2026-04-17

### 修复

- 修复在线封面自动获取失败的问题：各音乐 API（酷狗、网易云、QQ音乐）独立容错，单个 API 故障不影响整体搜索结果
- 修复音乐标签编辑后 UI 不刷新的问题：保存标签/歌词/封面后立即更新播放界面
- 修复歌词来源设置后不生效的问题：设置在线歌词来源后，如果正在播放该歌曲，歌词立即刷新
- 修复在线封面应用后不显示的问题：清除缓存并通知播放服务重绘
- 修复鼠标侧键无法作为自定义快捷键使用的问题：改为应用内 Listener 捕获（系统级 RegisterHotKey API 不支持鼠标按键）

### 改进

- 音乐标签编辑现在直接写入音乐文件元数据（ID3v2/VorbisComments/MP4Ilst），其他播放器也能看到修改
- 在线封面设置后同时写入音乐文件的封面标签
- 标签修改同步更新 index.json，确保重启/重建索引后修改不丢失
- 新增 Rust 侧 `write_tag_to_file` 和 `write_cover_to_file` API（基于 lofty 库）
- API 搜索结果增加空值保护，避免因 API 返回格式变化导致崩溃

## [1.6.0] - 2026-04-17

> Fork 后第一版发布，覆盖 `TASK_LIST.md` 的阶段一（稳定性修复）与阶段二（功能扩展）全部任务。

### 新增

- 关闭窗口默认最小化到系统托盘，托盘菜单支持：打开 / 上一首 / 播放暂停 / 下一首 / 退出
- 任务栏缩略图媒体控制按钮（上一首、播放/暂停、下一首），图标随播放状态动态切换
- 播放列表拖拽重排与自定义顺序持久化
- 歌曲播放次数统计、展示与排序
- 专辑碟号识别：多碟专辑按碟号 + 音轨号排列
- 歌词翻译显示开关：可在播放页 / 桌面歌词中控制是否显示翻译
- 音乐详情页右侧 A-Z 首字母索引（支持中文拼音首字母匹配）
- 音乐详情页"定位当前播放歌曲"按钮与当前播放高亮
- 播放页"添加到歌单"功能，支持直接创建新歌单
- 右键菜单"添加到歌单"支持新建歌单并添加当前歌曲
- 长歌名自动滚动显示
- 音量均衡功能（基于 ReplayGain 的自动音量补偿，可调预增益 -12dB ~ 12dB）
- CUE 文件支持（解析分轨、封面、歌词裁剪与时间轴重定位）
- 桌面歌词增强：字体选择、RGB 颜色输入、位置/颜色/锁定状态记忆
- 多选视图增强：Shift 范围选择，批量添加到歌单、批量删除（可选删除源文件）
- 快捷键系统重构：可自定义快捷键、支持鼠标侧键、后台可用范围控制
- 自定义背景图片
- 播放页无操作自动隐藏进度条及控件按钮（5 秒超时）
- 在线封面自动获取（仅对无封面歌曲生效）
- 音乐标签在线编辑（封面、歌词、Tag）
- WAV 格式元数据完善读取
- DTS 音频格式播放支持
- m3u 播放列表文件导入，自动生成对应歌单
- 多选功能入口优化：直接在歌曲列表右上角显示多选框
- 文件夹管理页面独立化，支持扫描音乐库
- 歌曲信息展示：播放页显示文件类型、采样率、比特率等详细信息
- 播放页歌曲删除功能

### 变更

- `desktop_lyric` 改为仓库内置依赖（`path: third_party/desktop_lyric`），确保构建可复现
- 静音默认快捷键从 `M` 调整为 `Alt + M`，避免与文本输入冲突
- 音量调节步进从 5% 调整为 1%
- 播放详情页控件自动隐藏时长从 3 秒调整为 5 秒
- 列表排序逻辑统一中英文行为，按拼音 / 字母一致排序
- 随机播放逻辑：上一首 / 下一首均在随机序列中跳转，避免回跳
- 默认音量从 100% 下调至 20%

### 修复

- 播放状态与 UI 图标（播放/暂停）不一致
- 桌面歌词开启后不立即显示当前正在播放的歌词行
- 多文件夹扫描导致相同歌曲重复显示（路径 + 元数据指纹双重去重）
- 搜索功能对歌手、专辑关联性不足
- 独占模式偶发不生效
- 歌曲标签乱码与严重显示异常
- 歌单持久化丢失（重启后歌单为空）
- 内置字体对常用字不生效
- 最大化后关闭再从托盘打开时最大化状态丢失
- WAV 元数据读取缺失
- 桌面歌词设置不记忆（字体、位置、颜色、锁定状态等）
- 桌面歌词字体选择弹窗空白与 RGB 设置界面异常
- 桌面歌词进程退出后不清理导致残留
- 自定义背景设置后不即时生效
- 中文歌曲首字母查找不准确
- 字母索引高亮与定位按钮显示异常
- 启动时音量恢复为 100% 的问题

### CI / 构建

- 新增并重构 Windows CI 工作流：
  - 主工程：`flutter analyze` → `cargo check` → `flutter test` → `flutter build windows --release`
  - `third_party/desktop_lyric`：独立 `flutter analyze` → `flutter build windows --release`
  - BASS 依赖自动下载与归档（含 `bass_aac.dll`）
- `analysis_options.yaml` 排除 `third_party/desktop_lyric/**`，避免第三方代码风格影响主仓质量门禁
